7. * Создать директорию, в которой есть несколько файлов. Сделать так, чтобы открыть файлы 
можно было, только зная имя файла, а через ls список файлов посмотреть было нельзя.



Create new folder and required files
_________________________________________________________________________
olegkrilov@olegkrilov-VirtualBox:~/les5/task5$ mkdir restricted-folder
olegkrilov@olegkrilov-VirtualBox:~/les5/task5$ ls -li
